## 코드 구조

### 세미콜론

자바스크립트는 줄 바꿈이 있으면 이를 '암시적'세미클론으로 해석한다.

이런 동작 방식을 세미콜론 자동 삽입(automatic semicolon insert)이라 부른다.

아래와 같이 줄 바꿈이 세미콜론을 의미하지 않는 경우도 있다.
~~~JavaScript
alert(3 +
1
+ 2);
~~~

## 주석

한줄 주석 : command + /

여러줄 주석 : command + option + /

> 프로덕션 서버에 배포하기 전에 코드를 압축해주는 도구가 많이 있고,
> 이 도구들은 주석을 삭제 해준다.
